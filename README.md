# TAJIKSET

Одностраничный каталог электроскутеров TAJIKSET.

## Возможности

- таджикский и русский языки;
- поиск и фильтры по назначению;
- блок популярных моделей;
- переключение каталога между сеткой и списком;
- кликабельные карточки и окно с характеристиками;
- системная кнопка «Поделиться моделью»;
- выбор аккумулятора для моделей №8 и №9;
- корзина с сохранением в браузере;
- сохранение данных покупателя и проверка формы;
- оформление заказа через WhatsApp;
- бесплатная доставка по Душанбе;
- согласование доставки по остальному Таджикистану;
- адаптация для телефона, планшета и компьютера.

## Запуск

Откройте `index.html` через Live Server или любой локальный HTTP-сервер.

Тестовый WhatsApp указан в `app.js`:

```js
const WHATSAPP_NUMBER = "992876477676";
```

Перед публикацией замените его на номер владельца магазина.
